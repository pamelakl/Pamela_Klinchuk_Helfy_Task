#Task Manager App

##Backend Setup
1. cd backend
2. npm install
3. npm start 

##Frontend Setup
1. cd frontend
2. npm install
3. npm start

##API Endpoints
-GET /api/tasks
-POST /api/tasks
-PUT /api/tasks/:id
-DELETE /api/tasks/:id
-PATCH /api/tasks/:id/toggle

##Assumtion
On the backend side, I did not implement a strict backend model or schema validation specifically for the update request's fields
